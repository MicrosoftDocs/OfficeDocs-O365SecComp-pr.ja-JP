# <a name="eu-gps-coordinates"></a>EU の GPS 座標

Office 365 のセキュリティでは、データ損失防止 (DLP)&amp;コンプライアンス センターには、DLP ポリシーに使用する準備ができている多くの機密性の高い情報の種類が含まれています。このトピックでは、GPS 座標を EU の機密情報の種類について説明します。
  
## <a name="austria"></a>オーストリア

### <a name="format"></a>形式

座標の範囲では、オーストリアの都市の: 緯度 46.526 から 48.816 と 16.945 に、9.6 からの経度です。
  
### <a name="pattern"></a>パターン

各座標、最大 6 桁の数字と小数点の位置の組み合わせです。
  
- 最大 6 桁の数字
    
- 最初または 2 番目の数字の後の 10 進ポイントです。

該当なし
