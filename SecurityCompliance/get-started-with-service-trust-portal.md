# <a name="get-started-with-the-microsoft-service-trust-portal"></a>Microsoft Service Trust Portal を始める。

Microsoft サービスの信頼のポータルでは、さまざまなコンテンツ、ツール、および Microsoft のセキュリティ、プライバシー、コンプライアンス ・ プラクティスに関するその他のリソースを提供します。
  
## <a name="accessing-the-service-trust-portal"></a>サービスの信頼のポータルへのアクセス

サービスの信頼のポータルには、マイクロソフトのコントロールおよびクラウド サービスと顧客データを保護するその中のプロセスの実装に関する詳細が含まれています。サービス信頼ポータル上のリソースのいくつかにアクセスするにする必要があります (Azure Active Directory 組織のアカウント、または Microsoft アカウント) マイクロソフト クラウド サービス アカウントで認証されたユーザーとしてログインし確認し、マイクロソフトがそのまま使用コンプライアンス資料の機密保持契約。
  
**既存のお客様がサービスを信頼のポータルにアクセスできる** [https://aka.ms/STP](https://aka.ms/STP) (試用版または有料) は、次のオンライン サブスクリプションのいずれか。 
  
- Office 365

- Dynamics 365

- Azure
    
 > [!NOTE]
 > Azure Active Directory アカウントが組織に関連付けられているドキュメントおよびコンプライアンス マネージャーのような機能の全範囲へのアクセスがあります。個人用に作成した Microsoft アカウントには、サービスの信頼のポータル コンテンツへのアクセスが制限されます。 
  
**新規のお客様およびお客様のマイクロソフトのオンライン サービスを評価します。**
  
新しいアカウントを作成する、または試用版のアカウントを作成するには、STP へのアクセスを取得する (試用版のアカウントにも使用)、次のサインアップ フォームのいずれかを使用します。
  
- 新しい[試用版のアカウントを Office 365 のビジネス](https://go.microsoft.com/fwlink/p/?LinkID=507653)、または新しい[Office 365 エンタープライズ試用版のアカウント](https://go.microsoft.com/fwlink/p/?LinkID=698279)にサインアップします。

- 新しい[Dynamics 365 試用版のアカウント](https://go.microsoft.com/fwlink/?LinkId=252780)にサインアップします。

- 新しい[Azure の試用版アカウント](https://go.microsoft.com/fwlink/?LinkId=722737)にサインアップします。 
    
無料試用版、またはサブスクリプションのいずれかにサインアップすると、STP へのアクセスをサポートするために Active Directory の Azure を有効にする必要があります。

## <a name="my-library"></a>ライブラリ

ライブラリのページにサービスを信頼のポータル上のドキュメントとリソースを追加するのには、ライブラリの機能を使用します。これにより、1 つの場所に関連するドキュメントにアクセスします。 マイ ライブラリにドキュメントを追加するには、 **...** ] メニューの [文書の右側をクリックし、**ライブラリに追加**] をクリックします。マイ ライブラリに複数のドキュメントを追加するには 1 つまたは複数のドキュメントの横のチェック ボックスをクリックし、ページの上部にある**ライブラリに保存する**で。

さらに、通知機能には、マイクロソフトは、マイ ライブラリに追加したドキュメントを更新するたびに、電子メール メッセージを送信するように、自分のライブラリを構成することができます。通知を設定するには、マイ ライブラリに移動し、**通知の設定**] をクリックします。通知の頻度を選択し、通知を送信するのには、組織の電子メール アドレスを指定できます。電子メール通知には、更新されたドキュメントへのリンクと更新プログラムの簡単な説明が含まれます。

通知を有効にするかどうかに関係なく、最後の 30 日以内に更新されたドキュメントが、ライブラリで特定するにも注意してください。更新プログラムの簡単な説明は、ツール ヒントにも表示されます。 

## <a name="starter-packs"></a>スターター パック

スターター パックは、特定の業界のマイクロソフトのクラウド サービスについてのドキュメントの Microsoft curated セットです。現時点では、サービスの信頼のポータルには、金融サービスの組織は、次の 3 つのスターター パックが用意されています。これらのスターター パックは、組織を評価し、セキュリティ、コンプライアンス、およびマイクロソフトのクラウドでのプライバシーを評価し、規制の厳しい金融サービス業界では、マイクロソフトのクラウド サービスの導入を支援するガイダンスを提供を支援します。

- **スターター パックの評価**- 金融サービス企業のマイクロソフトのクラウドの初期評価のために使用します。

- **スターター パックの評価**- 評価後を使用して、チェックリスト、およびその他のガイダンスこのスターター パックのセキュリティ、コンプライアンス、およびプライバシーに関連するリスクを評価する組織を支援します。

- **監査のスターター パック**のユーザーを支援する方法で、マイクロソフトのクラウド サービスの実装に役立つ監査制御およびその他のツールを使用する方法については、このスターター パックは、リスク、組織のリスクを軽減します。

これらのスターター パックにアクセスするには、**サービス信頼ポータル _gt 業界 & 領域 _gt 業界ソリューション _gt 金融サービス**に移動します。開くか、またはスターターからのダウンロード文書をパックまたは、マイ ライブラリに保存します。
